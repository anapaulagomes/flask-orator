Flask-Orator
============

Flask-Orator adds `Orator ORM <https://github.com/sdispater/orator>`_ support to Flask applications.

Since it is merely a wrapper for Orator, it has all its benefits:

.. toctree::
    :maxdepth: 2

* A simple but powerful :ref:`ORM`
* A database agnostic :ref:`SchemaBuilder`
* A low level :ref:`QueryBuilder` to avoid the overhead of the ORM
* :ref:`Migrations`
* Support for **PostgreSQL**, **MySQL** and **SQLite** out of the box

Flask-Orator supports python versions **2.7+** and **3.2+**
